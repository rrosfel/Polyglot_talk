# Conversational app to learn languages with three distinct AI teachers and receive feedback


To view this app in AI Studio: https://ai.studio/apps/drive/1g0WfGhcM8839tjvqak5-LzJAOng-e0DR

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
